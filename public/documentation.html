MyFlix

This is a web application that allows users to search for information about different movies, directors, and genres. Users can sign up for an account, update their account information, and even personalize a list of their favorite movies!

MyFlix's Features

  • Return a list of ALL movies to the user
  • Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user
  • Return data about a genre (description) by name/title (e.g., “Thriller”)
  • Return data about a director (bio, birth year, death year) by name
  • Allow new users to register
  • Allow users to update their user info (username)
  • Allow users to add a movie to their list of favorites
  • Allow users to remove a movie from their list of favorites
  • Allow existing users to deregister

API Endpoints

Business Logic URL HTTP Method Query Parameters Request Body Data Format Response Body Data Format
Get a List of all Movies /movies GET - - A JSON object holding data about all the movies
Get a Single Movie by Title /movies/:title GET :title - A JSON object holding data about a single movie
Get Genre by Name /movies/genre/:genreName GET :genreName - A JSON object holding data about the genre of a movie
Get Director by Name /movies/directors/:directorName GET :directorName - A JSON object holding data about the name of a director
Allow New Users to Register /users POST - A JSON object holding data about the users to add, structured like this { "name" : "Keith", "favorite movie" : [] } A JSON object holding data about the user that was added, structured like this { "name" : "Keith", "favorite movie" : [] } including an ID
Allow New Users to Update their User Information /users/:id PUT :id - A JSON object holding data about the updated information
Adds Movie to Favorite List /users/:id/:movieTitle POST :movieTitle - A text message indicating whether the movie was added
Deletes Movie from Favorite List /users/:id/:movieTitle DELETE :movieTitle - A text message indicating whether the movie was removed
Allow Exisiting Users to Deregister /users/:id DELETE :movieTitle - A text message indicating whether the User has deregistered
