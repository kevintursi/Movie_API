MyFlix

This is a web application that allows users to search for information about different movies, directors, and genres. Users can sign up for an account, update their account information, and even personalize a list of their favorite movies!

MyFlix's Features

  • Return a list of ALL movies to the user
  • Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user
  • Return data about a genre (description) by name/title (e.g., “Thriller”)
  • Return data about a director (bio, birth year, death year) by name
  • Allow new users to register
  • Allow users to update their user information
  • Allow users to add a movie to their list of favorites
  • Allow users to remove a movie from their list of favorites
  • Allow existing users to deregister

API Endpoints

Get All Users /users GET - - A JSON object holding data about all the users
Get a User by Username /users/:Username GET - - A JSON object holding data of a specific user
Get a List of all Movies /movies GET - - A JSON object holding data about all the movies
Get a Single Movie by Title /movies/:Title GET :Title - A JSON object holding data about a single movie
Get Genre by Name /movies/genre/:genreName GET :genreName - A JSON object holding data about the genre of a movie
Get Director Information by Name /movies/directors/:directorName GET :directorName - A JSON object holding data about a director
Allow New Users to Register /users POST - A JSON object holding data about the users to add, structured like this { "Username" : "String", "Password": "String", "Email": "String, "Birthday": "Date"} A JSON object holding data about the user that was added, structured like this { "Username" : "Henry", "Password": "Password5", "Email": "Henry@gmail.com", "Birthday": "2000-01-01", "Favorite Movies": [] } including an ID
Allow New Users to Update their User Information /users/:Username PUT :Username - A JSON object holding data about the updated information
Adds Movie to Favorite List /users/:Username/movies/:MovieID POST :Username, :MovieID - A JSON object holding data about the User including the movie being added to User's Favorite Movies
Deletes Movie from Favorite List /users/:Username/movies/:MovieID DELETE :Username, :MovieID - A JSON object holding data about the User and showing the movie no longer on the User's Favorite Movie list
Allow Exisiting Users to Deregister /users/:Username DELETE :Username - A text message indicating whether the User has deregistered
